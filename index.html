LAST STAND

WELCOME TO LAST STAND!!

The aim of this game is simple, SURVIVE!! against a relentless horde of zombies for 3.5 minutes.

We are not so ruthless to leave you stranded withuot any supplies during a zombie apocalypse.

You are given defense items in the inventroy which you use to set your base around the cannons, YES CANNONS!! they are there to protect you.

The cannon shoot on eihter side at 10s of interval automatically. Good Luck Survivor!!

Controls:

Change Guns : Press the number keys to change guns, '1' :M4A1, '2' :AWM, '3' :AKM, '4' :Desert Eagle, '5' :UZI. (Click here to view Gun Stats)

Movements : 'A' :Move left, 'D' :Move right, 'K' :Jump, 'left click' :To shoot & 'R' :To reload the gun

Inventory : 'I' :opens inventory, where you can select and deploy defense items like Defense Blocks, Spike Traps and Land Mines.(Beware: the iventory can only be accessed during the preparation phase which is the first 30s of the game)

Shop : 'S' :opens shop where you can use your score to purchase power ups and ammo.

JetPack : 'J' :toggles jetpack, Beware: it'll only work as long as you have enough fuel, so use jetpack wisely. The jetpack will refuel automatically as time passes.
